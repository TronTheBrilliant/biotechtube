# BiotechTube UX Audit Report
**Date:** March 22, 2026
**Scope:** Full platform review — all public pages, desktop + mobile

---

## Executive Summary

BiotechTube has strong bones: clean visual design, consistent theming, solid data architecture, and a professional feel that punches above its weight for an early-stage platform. The new top navigation is clean and the design language (CSS custom properties, teal accent, Geist font) is cohesive.

However, the platform has critical credibility issues (conflicting data claims), one broken page (/people), several dead-end user flows, and is missing the conversion mechanics needed to turn visitors into users. The gap between what the site *promises* and what it *delivers* will erode trust with the sophisticated biotech/investor audience.

**Overall Grade: C+** — Great foundation, needs polish and honesty.

---

## 🟢 What's Good

### 1. Visual Design & Consistency (A-)
- **Color system is excellent.** CSS custom properties with light/dark mode support, teal accent that feels medical/scientific without being sterile.
- **Typography is strong.** Geist Sans is modern and readable. Font scaling is well-calibrated (11px labels → 36px headings). Space Grotesk for brand elements adds character.
- **Component consistency.** Every card, badge, button, and table uses the same design language. No component feels like it belongs to a different site.
- **Emoji usage is smart.** The 🧬💰📊🔥 icons throughout give quick visual scanning cues without feeling childish — they work for this audience.

### 2. Homepage Layout (B+)
- **Two-column layout with sidebar** works well — main content + contextual info (recently funded, events).
- **Tab bar** (📈 Top / 🔥 Trending / 💰 Funded / 🆕 New / ⭐ Watchlist) is a strong pattern — gives users 5 different lenses on the same dataset.
- **Country/region dropdown** is well-organized with 7 regional groupings.
- **RankingTable** is dense but readable — good use of horizontal space.

### 3. Company Profiles (B)
- **Multi-tab structure** (Overview / Pipeline / Report / Funding / Team / News) is the right pattern for this much data.
- **AI-generated company overviews** add real value — no competitor offers this depth for 14K+ companies.
- **Pipeline visualization** with stage progression bars is intuitive.
- **Funding timeline** with dots and connecting lines tells a story.

### 4. Navigation Redesign (B+)
- **Top bar** is clean: logo, 4 menu triggers, search shortcut (⌘K), pricing, sign in.
- **Dropdown menus** with colored icon boxes and subtitles are well-designed.
- **Featured sections** in dropdowns (Top Countries, Therapeutic Areas) provide useful shortcuts.
- **Mobile accordion** is functional with one-at-a-time expansion and proper scroll.

### 5. Technical Foundation (A-)
- **Keyboard shortcuts** (⌘K for search, ESC to close).
- **Debounced search** (200ms) feels responsive.
- **ARIA attributes** present on nav triggers.
- **prefers-reduced-motion** respected for animations.
- **Dark mode** fully implemented in CSS (even if toggle isn't prominent).

## 🔴 Critical Issues

### 1. Data Credibility Crisis — MOST DAMAGING ISSUE
**The site contradicts itself on nearly every page.**

| Claim | Location | Reality |
|-------|----------|---------|
| "14,000+ companies" | Homepage hero, About, Nav | Therapeutic areas page totals ~693 across all categories |
| "10,000+ companies" | Homepage pills | Conflicts with 14,000 claim on same page |
| "$4.2B YTD funding" | Markets page | Funding page shows only $340M tracked |
| "30+ countries" | Homepage | Funding page filters limited to 4 countries |
| "10,632+ companies" | Homepage stat | Third different number on the same page |

**Why this is critical:** Your target audience is investors, analysts, and BD professionals. These people spot data inconsistencies for a living. If the homepage can't get its own numbers straight, they won't trust the data inside.

**Fix:** Pick accurate numbers. If you have 14,000 company *records* in Supabase but only ~700 with rich profiles, say "14,000+ company profiles, 700+ with detailed intelligence." Honesty builds more trust than inflated claims.

### 2. /people Page is Broken (F)
- Page exceeds **10MB** — all entries render at once with zero pagination or virtualization.
- Likely crashes or freezes on mobile devices and slower connections.
- This is the single worst technical issue on the platform.

**Fix:** Server-side pagination (50 per page), alphabetical letter navigation should filter server-side, not client-side rendering of thousands of DOM nodes.

### 3. About Page Contact Emails Are Dead
```tsx
<span style={{ color: "var(--color-accent)" }}>
  hello@biotechtube.com
</span>
```
These are `<span>` elements, not `<a href="mailto:...">` links. Users cannot click to email. The email addresses *look* clickable (teal colored) but aren't.

**Fix:** Wrap in `<a href="mailto:hello@biotechtube.com">`.

### 4. No Primary Conversion CTA on Homepage
The most important page has no "Sign Up" or "Get Started" button above the fold. The only signup path is buried in the nav ("Sign in") and footer. For a platform trying to convert visitors to users, this is a major gap.

**Fix:** Add a prominent CTA button in the hero section: "Start free →" with accent background.

## 🟡 Significant Issues

### 5. Pipeline Content is Embarrassingly Thin
The Pipeline page — a core value proposition — tracks only **10 drug programs across 8 companies**. For a platform claiming to track "10,000+ companies," having 10 pipeline entries is not credible.

**Fix:** Either bulk-populate pipeline data from public sources (ClinicalTrials.gov, SEC filings) or de-emphasize pipeline in marketing until the data is there. Don't put it as a primary nav item if it's nearly empty.

### 6. "Similar Companies" Algorithm is Broken
On the Novo Nordisk profile (a $162B+ company), "Similar Companies" shows tiny pre-clinical startups. Showing OncInvent alongside Novo Nordisk destroys the feature's credibility.

**Fix:** Weight similarity by market cap tier, stage, and therapeutic area. A large-cap company should show other large-caps in similar therapeutic areas.

### 7. Companies Page — Empty "Recently Added" Section
The `/companies` gateway page has a "Recently added" header with zero content beneath it. Empty sections make the platform look abandoned.

**Fix:** Either populate it from the database (companies with most recent `created_at`) or remove the section.

### 8. Investors Page — Non-functional Filters + Misleading Data
- Therapeutic area filter tags appear to do nothing when clicked.
- "Public shareholders" is listed as a single investor entity with **$4,482.5B** in combined portfolio value — this is a data aggregation artifact, not a real investor.
- No way to distinguish VCs from institutional investors from public shareholders.

### 9. Funding Page — Geographic Blindspot
Despite "Global" positioning, funding page filters are limited to **4 countries** (Norway, Sweden, Denmark, Germany). No US, UK, China, Japan, or any other major biotech market.

### 10. No Loading States Anywhere
No skeleton screens, no shimmer effects, no spinners (except search). When data tables load, users see nothing → everything at once. This feels jarring, especially on slower connections.

**Fix:** Add skeleton loaders for RankingTable rows (gray rectangles that pulse). Even fake 200ms delays with skeletons feel faster than instant pop-in.

## 🟠 Moderate Issues

### 11. Pricing Page Missing Standard Elements
- No annual billing toggle (industry standard, usually offers 20% discount)
- No FAQ section answering objections
- No social proof (customer logos, testimonial quotes, user counts)
- All three tiers' CTAs say "Get started" — should differentiate (e.g., "Start free" / "Start trial" / "Contact sales")

### 12. Search Could Be More Powerful
- Search only covers companies — no results for drugs, people, investors, or therapeutic areas.
- No autocomplete suggestions while typing.
- No recent searches or search history.
- No "trending searches" to help new users discover value.

### 13. No Breadcrumb Navigation
Users landing on deep pages (company profiles, drug pages, investor pages) have no orientation breadcrumbs. On a data platform with 4+ levels of hierarchy, breadcrumbs are essential.

### 14. Footer Links to Non-existent/Hollow Pages
- `/sponsors` — likely returns 404
- `/templates` — likely returns 404
- `/submit-pitch` — unclear if functional
- Footer should only link to pages that exist and work.

### 15. No Dark Mode Toggle Visible
The CSS fully supports dark mode (`.dark` class with inverted variables) but there's no visible toggle for users. The theme provider is set to `defaultTheme="light"` with `enableSystem={false}`. This means dark mode code exists but is completely inaccessible.

### 16. No Error Pages
No `error.tsx` or `not-found.tsx` custom pages found. Users hitting bad URLs get the default Next.js error page, which breaks the brand experience.

## The Big Picture

BiotechTube's core problem isn't design — it's **credibility gap**. The site looks like it tracks 14,000 companies but the data underneath tells a different story. The design is genuinely good (top 20% of SaaS sites), but the content doesn't match the promise.

**The path to genius:**
1. **Be honest about what you have** — "Detailed profiles on 700+ biotech companies, with basic listings for 14,000+" is more trustworthy than "14,000+ companies" when most profiles are thin.
2. **Fix the broken stuff first** — /people page crash, dead email links, empty sections. These are trust-killers.
3. **Add conversion mechanics** — CTAs, social proof, email capture. Beautiful pages that don't convert are just art.
4. **Then go deep** — bulk data import, cross-entity interlinking, personalized dashboards.

The foundation is excellent. The design language is professional and consistent. With data honesty and UX polish, this platform can genuinely compete with players charging 100x more.
